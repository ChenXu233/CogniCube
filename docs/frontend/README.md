# 说明

前端基于React+TypeScript+React Native

相关的文档如下：
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [React Native](https://reactnative.dev/)

## 使用流程

1. 进入目录

```bash
cd ./frontend/
```

2. 确保已经安装nodejs

```bash
node -v
```

3. 安装依赖

```bash
npm install
```

4. 使用指令运行

```bash
npm run start
```
